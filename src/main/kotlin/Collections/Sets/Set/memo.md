## Set
重複機能をサポートせず、順序も持たないコレクション。

## Properties
- size
  - 要素数を返す。

## Functions
- contains
  - 指定した要素が含まれているかどうかを返す。
- containsAll
  - 指定した要素がすべて含まれているかどうかを返す。
- isEmpty
  - 要素が空かどうかを返す。
  
## Extension Properties
- indices
  - インデックスの範囲を返す。
- all
  - すべての要素が条件を満たすかどうかを返す。
- any
  - いずれかの要素が条件を満たすかどうかを返す。
